# Usage

# QuickCSS
```css
:root {
  /* Default HSL values */
  --default-hue: 0;
  --default-saturation: 0%;
  --default-lightness: 80%;
  --default-alpha: 0.2;

  /* Self HSL values */
  --self-hue: 285;
  --self-saturation: 70%;
  --self-lightness: 65%;
}
```
